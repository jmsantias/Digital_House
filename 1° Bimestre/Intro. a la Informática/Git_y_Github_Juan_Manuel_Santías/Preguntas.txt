a. ¿Quién inventó el sistema de control de versión Git y por qué?
  a- El sistema fue inventado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

b. ¿A quién pertenece actualmente Github y por qué?
  b- El 4 de junio de 2018, Microsoft compró GitHub. Esto fue porque GitHub es la plataforma más importante de colaboración para proyectos.

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
  c- Se puede trabajar descargando la aplicación de GitHub a la pc.
